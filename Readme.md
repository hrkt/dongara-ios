# Dongara iOS

A boilerplate project for wrapping web as a native app.

## Feature

- Run as a native iOS app which shows the web site 

## How to use

1. change the property "AppUrl" in the dongara/App.plist

![Screenshot](doc/app-url.png?raw=true "property")

2. Run
